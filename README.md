# Rageman-Orders

Ini adalah aplikasi kasir & waiter untuk resto & coffee,
dibuat saat magang / kuliah.

## Teknologi
- PHP
- JavaScript
- MySQL
- HTML/CSS
- Bootstrap

## Cara Jalankan
1. Clone repo
2. buka dengan XAMPP
3. akses di localhost
